Terms and privacy

The data/personal information (login credentials and all saved passwords) are encrypted and stored into a secured database and nothing will be done with them (except for sending the password reset email).

So, every information inserted into this web application are touched only to secure them.

Services

This web application offers some simple tools to manage your passwords.

In details, you can:

  • Login and signup to manage your passwords
  • Reset your account password if you forget it or want to change it
  • Create, Read, Update, Delete password
  • All the passwords created include those informations:
    • _id: created automatically, used to identify the password
    • name: a name assigned to recognize the password required
    • login: the login information to sign into the website (tipically email or username)
    • secondLogin: if one website have a second login information
    • password: the password to access into the website
    • url: url of the password's website
    • notes: some notes (e.g: a pin is required after login, you can store it here)
    • userId: the user id whos belongs the password
    • date create and date update: the creation/update date of the password
  • Import / export the passwords
  • Delete your account and all your data
